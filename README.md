# liftz Workout Tracker App

[liftz](https://liftz-workout-tracker.vercel.app/) is a web application is designed to help users efficiently track and manage their workouts.

This was the first time I built a workout tracker using the MERN stack. I've recently built a newer logging app called [FitFlash](https://fitflash.vercel.app/) using Next.js + Supabase to avoid having to wait for backend server cold starts.

## Technologies

- React (Next.js, Chakra UI, Formik, Yup)
- Express (Passport.js, Express Validator)
- Node.js
- MongoDB (Mongoose)

## Features

- Workout creation, logging, and session tracking
- Workout-based and exercised-based analytics
- Barbell calculator
